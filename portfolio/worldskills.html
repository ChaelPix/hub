My WorldSkills Journey



I enrolled in WorldSkills from October 2022 to September 2024.

I participated in Autonomous Mobiles Robotics skill.

I had to create and program autonomous robots with ROS.

Lyon2024 is the 5-days Worlds final competition. I competited against 21 other countries.

The robot was programmed with ROS2 Humble using C++ and Python. Tasks were:

    Navigation with Lidar: Used SLAM, AMCL, and Nav2.
    Object Detection: Used a DeepSense camera with YOLOv8 for detection and TF transformations to convert image position to robot-relative position.
    Object Manipulation: Built a 3-segmented arm and used Track_IK for inverse kinematics.
    Sowing: Hopper system using a servo-motor.

Additionally, I developed a real simulation in Unity using C# that interfaces with ROS2.

SkillsAustria is the 3-days national competition of Austria at Salzburg where I was invited to participate as part of my training after being selected for the French WorldSkills Team.

The robot was programmed with ROS2 Humble using C++. Tasks were:

    Navigation with IR and Ultrasonics sensors: Used SLAM, AMCL, and Nav2.
    QR code reader: Used a raspberry camera with OpenCV.

This was the 4-days national final competition at Lyon. I had the 2nd place because of some robot hardware issues.

The robot was programmed with ROS1 Noetic using C++. Tasks were:

    Navigation: Used a Lidar with Hector SLAM.
    QR code reader: Used a Pixi2 camera with OpenCV.

This was the 3-day regional competition in Nouvelle-Aquitaine. I enrolled in the competition on the morning it started. I learned everything during the nights.

The robot was programmed with Arduino C++. The tasks were:

    Navigation on sand without tank tracks.
    Navigation in a small maze.
    Color follower: Used a Pixy2 camera.
