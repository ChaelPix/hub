RN41

Suites

Méthodologie pour l'étude des suites

1. Types de suites et comportements

  1. Suite arithmétique

.. math:: u_{n+1} = u_n + r

Exemple : uₙ = 2n + 3
- u₀ = 3
- u₁ = 5
- u₂ = 7
→ Diverge vers +∞ car r > 0
  1. Suite géométrique

.. math:: u_{n+1} = qu_n

u_n = u_0 q^n

Exemple : avec q = 1/2
- u₀ = 1
- u₁ = 1/2
- u₂ = 1/4
→ Converge vers 0 car |q| < 1
  1. Suite récurrente

.. math:: u_{n+1} = f(u_n)

Exemple : f(x) = 1/x, u₀ = 2
- u₀ = 2
- u₁ = 1/2
- u₂ = 2
→ Oscillante car f∘f(x) = x

2. Techniques de calcul essentielles

  1. Étude de la monotonie

Pour :math:u_n = \frac{n^2+1}{n} :

.. math:: u_{n+1} - u_n &= \frac{(n+1)^2+1}{n+1} - \frac{n^2+1}{n} \ &= \frac{n-1}{n(n+1)} \ &> 0 \text{ pour } n > 1

  1. Calcul de limite

Pour :math:u_n = \frac{n^2}{n^2+1} :

.. math:: u_n = \frac{1}{1 + \frac{1}{n^2}} \xrightarrow[n \to +\infty]{} 1

  1. Points fixes et convergence

Pour :math:u_{n+1} = \frac{u_n + 2}{3} :

.. math:: l = \frac{l + 2}{3} \implies l = 1

3. Théorèmes clés

  1. Convergence

    • Suite croissante majorée ⟹ convergente
    • Suite décroissante minorée ⟹ convergente

  2. Critère du quotient

.. math:: \left|\frac{u_{n+1}}{u_n}\right| < 1 &\implies \text{converge vers 0} \ \left|\frac{u_{n+1}}{u_n}\right| > 1 &\implies \text{diverge}
