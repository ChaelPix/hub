For an English class assignment, we had to imagine a video game concept for a presentation. Instead of just presenting an idea, my friend and I decided to actually build the game in 24 hours for the next class. We developed it using Unity with C#, and also set up a SQL database connected to a PHP API.

The game is playable on my friend's website: Super Nova Sprint
