==========
Contribuer
==========

Vous trouverez ici une aide sur comment contribuer à cette documentation avec des explications sur
le fonctionnement du site, son architecture et les technologies utilisées.

- la technologie utilisée
- comment la build en local
- le déploiement sur GitHub
- son architecture
- comment contribuer

``reStructuredText``, le langage utilisé pour écrire le code source de ce site.
