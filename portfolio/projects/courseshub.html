As an engineering student, I learn a lof of things and I need to store every knowledge I get. I created this website to help me store all the knowledge I get from my classes and some from the internet in order to have a quickly access to it.


Visit the website

Hosted on GitHub Pages, using GitHub Actions.

Built with Sphinx, this project combines .rst and .md files to create clean, accessible documentation. I chose this approach because it lets me focus on the content while maintaining a professional look and excellent searchability.

What started as simple class notes has grown into a comprehensive knowledge base that I regularly update and refer to. It's become an invaluable tool in my learning journey.
