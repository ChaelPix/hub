This was a school project where my classmate and I had three weeks to develop a multiplayer battleship game from scratch using C++. The project included a TCP server that handled matchmaking. We decided to go further by adding features like a graphical interface, an online database to save stats, avatar customization, an internal game chat, and more.

I made the graphic interface and networking system.

The game can be downloaded on Itch.io: Itch.io Valiant
