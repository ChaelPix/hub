What is NEAT?

NEAT (NeuroEvolution of Augmenting Topologies) is a genetic algorithm that evolves both the weights and structure of artificial neural networks. Key concepts:

  • Genome: The complete genetic encoding of a neural network, including:

    • Nodes (neurons)
    • Connection genes (synapses with weights)
    • Innovation numbers (tracking structural mutations)

  • Species: Groups of similar genomes that compete within their niche

  • Innovation: Tracking system for new network structures

  • Crossover: Combining genetic material from two parent networks

  • Mutation: Random modifications including:

    • Weight mutations (Â±20% of current value)
    • Add/remove connections
    • Add/remove hidden nodes

Neural Network Architecture

Input Layer (5 neurons):

  • Bird Y-position
  • Vertical velocity
  • Horizontal distance to next pipe
  • Top pipe Y-position
  • Bottom pipe Y-position

Output Layer (1 neuron):

  • Flap probability (threshold 0.5)

Hidden Layer:

  • Dynamically evolves through mutation
  • Can develop hidden neurons

Precise Fitness Function

Total Fitness = Base Score + Bonuses - Penalties

  • Base Score:

    • +10.0 points per pipe passed
    • +0.1 points per frame survived

  • Position Bonuses:

    • Up to +1.0 point per frame for optimal pipe gap alignment
    • Bonus = max(0, 1 - |birdY - gapCenterY| / 150)

  • Penalties:

    • Instant death for hitting screen boundaries
    • Fitness reset to 0 on death

Performance Metrics

From a test run (Generation 0-17):

  • Initial Performance:

    • Gen 0: Random behavior, 0% survival rate
    • Gen 1: 7 birds reach first pipe, max fitness 137.34

  • Learning Progress:

    • Gen 9: Major breakthrough (207 pipes, fitness 12,381.09)
    • Gen 13-14: Peak performance (300 pipes, fitness 19,560.28)
    • Population adaptation: 80%+ birds consistently reaching first pipe

  • Mutation Impact:

    • ~275 mutations per generation
    • Optimal mutation rate: 30% chance per offspring
    • Top 75 birds preserved unchanged each generation
